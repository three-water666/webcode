# v0.8.3 (2026-05-25)

### Fixes
- **Result Delivery**: Verify tool result writes before auto-sending, retry failed input writes, and skip auto-send when delivery cannot be confirmed.
- **Duplicate Delivery Prevention**: Finalize unverified or failed delivery batches so the bridge does not repeatedly append the same result or paste oversized attachments.
- **Notification Focus**: Keep the MV3 service worker alive during notification click handling, activate the source tab first, and restore minimized windows before focusing them.

### Documentation
- **Browser Modes**: Added a bilingual browser mode guide and refreshed README guidance for default, installed, user-profile, and isolated browser launches.
- **Built-in Tools**: Added an English built-in tools reference and linked it from the English README.
