# v0.8.2 (2026-05-24)

### Features
- **Attention Alerts**: Added browser-backed completion and approval alerts, including an optional short sound cue when attention is needed.
- **Isolated Edge Launch**: Added quick launch support for a dedicated isolated Edge profile with the bridge extension loaded.

### Improvements
- **Browser Notifications**: Unified completion and approval notifications through the browser bridge, and focus the source AI tab when a notification is clicked.
- **Skill Loading**: Return workspace-relative skill file paths from `list_skills`, so agents can read skill files directly.

### Fixes
- **Completion Detection**: Hardened task completion detection for hidden send and stop button changes, config resets, and duplicate notification bursts.
- **Browser Windows**: Reuse keepalive browser windows more reliably.
- **Platform Defaults**: Added the missing inline character fallback for VS Code platform selectors and removed stale selector limits.

### Engineering
- **Background Services**: Split browser background services into smaller modules and hid bootstrap-only tools from model-visible tool lists.
- **Gateway Tests**: Split Node unit tests from extension host tests and ignored the VS Code test cache.
