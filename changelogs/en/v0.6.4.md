# v0.6.4 (2026-05-14)

### Features
- **Logger Controls**: Added a larger floating process log panel with clear, minimize, restore, close, drag, and resize controls.
- **Summary Filter**: Added a dedicated summary log stream that captures each tool call name and purpose for faster review.

### Improvements
- **Logger Filtering**: Added per-type filters for summary, info, success, warning, error, and action logs, with the standard log types toggleable as a group.
- **Popup State Sync**: Kept the browser popup log toggle synchronized when the floating log panel is closed from the page.

### Fixes
- **Summary Visibility**: Kept the summary filter independent from the all-standard-logs toggle so summaries remain easy to isolate.
- **HITL Approvals**: Suppressed duplicate browser notifications for approval prompts while keeping the in-page confirmation flow.

### Engineering
- **Development Setup**: Ignored the local pnpm store directory to keep generated dependency cache files out of git status.
