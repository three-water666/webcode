# v0.9.4 (2026-06-11)

## Features

- Add a popup tool auto-approval toggle. When enabled, tool approval checks are
  bypassed for the current browser tab session, reducing interaction steps. (#53)
- Scope the popup auto-send setting to the current browser tab session.
  Previously this setting was shared across all tabs via sync storage; it now
  uses tab-level state shared between the popup, background, and content scripts.
- Add execute command result summary. After a background command completes, the
  AI now receives a concise summary of stdout, stderr, exit code, and shell
  information, making it easier to interpret command results without reading
  raw output. (#53)
