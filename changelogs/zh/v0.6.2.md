# v0.6.2 (2026-04-25)

### 新增
- **项目品牌**：围绕 `webcode` 产品名完成浏览器插件、VS Code Gateway、包元数据、文档与发布产物命名更新。
- **初始化提示词快捷触发**：新增在聊天输入框中输入 `/webcode` 或 `@webcode` 后询问是否替换为初始化提示词的流程，替换后不会自动发送。
- **工具调用状态徽标**：为捕获到的工具调用 JSON 块增加本地化状态徽标，使执行中、执行完成和错误状态更醒目。

### 改进
- **初始化引导**：明确初始化提示词与 `webcode_init` 返回结果中的说明，降低模型在完成初始化后重复发送初始化命令的概率。
- **文档**：更新 README 与浏览器插件文档，说明新的 `/webcode` 与 `@webcode` 初始化流程。

### 修复
- **浏览器插件打包**：暴露生成后的浏览器插件 chunk，确保 content script 能正确加载构建产物。
- **初始化触发匹配**：修复 `/webcode` 与 `@webcode` 紧贴非 ASCII 任务文本时无法触发的问题，同时继续避免 URL 与 handle 场景误触发。
- **HITL 队列**：修复同批队列中的命令在第一个相同命令被永久允许后，后续命令仍重复弹出确认框的问题。
