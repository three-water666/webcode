# v0.6.3 (2026-05-13)

### 新增
- **命令安全**：新增 shell 命令风险检测，在执行前拒绝危险命令模式。
- **工具调用反馈**：新增执行中 loading 动画，并对齐 ChatGPT 工具调用状态徽标，让页面内执行状态更清晰。

### 改进
- **命令执行**：统一命令 shell 处理，并改进终端进程清理，使停止会话时能更可靠地终止子进程。
- **浏览器桥接**：提升浏览器插件开发模式稳定性、热更新行为，并完善自动发送相关 DOM 查询类型。
- **工具调用解析**：改进模型工具调用 JSON 畸形内容修复，并支持对缺少 request ID 的工具调用进行去重。

### 修复
- **自动发送**：修复浏览器端重复自动发送的问题。
- **推理块过滤**：忽略 DeepSeek 与 GLM 的推理 / 工具思考块，避免被误识别为可执行工具调用。
- **VS Code 诊断**：清理扩展调试诊断输出与相关开发配置。

### 工程
- **发布流程**：新增基于 tag 触发的 GitHub Release 工作流，将双语 changelog 按版本拆分，并补充 Windows PowerShell 发版构建路径说明。
- **质量检查**：加强 ESLint 覆盖范围，并在工作区内应用自动 lint 修复。
