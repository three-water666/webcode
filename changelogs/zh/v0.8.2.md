# v0.8.2 (2026-05-24)

### 功能
- **注意力提醒**：新增由浏览器承载的完成和审批提醒，并在需要注意时支持播放短提示音。
- **隔离 Edge 启动**：新增专用隔离 Edge 配置的快速启动能力，并自动加载 bridge 扩展。

### 改进
- **浏览器通知**：统一通过 browser bridge 发送完成和审批通知，并在点击通知时聚焦来源 AI 标签页。
- **技能加载**：`list_skills` 返回工作区相对的技能文件路径，方便 agent 直接读取技能文件。

### 修复
- **完成检测**：强化隐藏 send/stop 按钮变化、配置重置和重复通知突发场景下的任务完成检测。
- **浏览器窗口**：更可靠地复用 keepalive 浏览器窗口。
- **平台默认值**：为 VS Code 平台选择器补齐缺失的内联字符回退，并移除过期的选择器限制。

### 工程
- **后台服务**：拆分浏览器后台服务模块，并从模型可见工具列表中隐藏仅用于启动初始化的工具。
- **Gateway 测试**：拆分 Node 单元测试和扩展宿主测试，并忽略 VS Code 测试缓存。
