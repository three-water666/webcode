# v0.5.0 (2025-12-12)

**重大更新**：架构重构与核心功能增强。

### 新增
- **Architecture**：浏览器插件重构为 Vite + TypeScript Monorepo 结构。
- **Performance**：实现工具分层懒加载（Tiered Tool Discovery），大幅降低 Token 消耗。
- **Sync**：实现基于 Host 的配置云同步能力（Host-based Config Sync）。
- **Options**：新增用户规则（User Rules）配置，支持自定义个性化指令。
- **Adapters**：新增对 AI Studio 的支持，并更新页面选择器。

### 修复
- **Bridge**：修复自动发送流程中的竞态死锁问题。
- **Bridge**：修复发送逻辑 Bug 与设置页配置持久化问题。
